Suppliers Manager for Woocommerce allows you to manage your suppliers and associate Woocommerce products to them. With the PRO version of this plugin, you can order Products when stocks are getting low.

== Description ==
With Suppliers Manager for Woocommerce, you can :

* List your suppliers with extra informations (address, contact, etc.)
* Link Woocommerce products with suppliers
* Follow product stocks for each supplier

Please, see our PRO version for more features : [Suppliers Manager for Woocommerce PRO](https://wp-shopping.com/suppliers-manager-pour-woocommerce/?utm_source=plugin&utm_campaign=bouton_plugin)

== Installation ==
* Upload the plugin files to the `/wp-content/plugins/suppliers-manager-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
* Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==
* How can I add a supplier to Woocommerce?
Go in the 'Supplier' section and click on the 'New' button.

* How can I attach products to a supplier?
Go on the product page and set 'supplier' field. You can find it in the product datas metabox.
You can also set the supplier for one or many products by using the products table bulk action 'set supplier'.

* Does this plugin work with all WordPress versions?
This version works with WordPress 4.6 and better. If you're using an older version, some bugs can happen.
